========================================================================
MICROSOFT 基础类库: SOR 项目概述
========================================================================


应用程序向导已为您创建了此 SOR DLL。此 DLL 不仅演示 Microsoft 基础类的基本使用方法，还可作为您编写 DLL 的起点。

本文件概要介绍组成 SOR DLL 的每个文件的内容。

SOR.vcproj
这是使用应用程序向导生成的 VC++ 项目的主项目文件。 
它包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

SOR.h
这是 DLL 的主头文件。它声明了 CSORApp 类。

SOR.cpp
这是主 DLL 源文件。它包含 CSORApp 类。

SOR.rc
这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。

res\SOR.rc2
此文件包含不在 Microsoft Visual C++ 中进行编辑的资源。您应该将不可由资源编辑器编辑的所有资源放在此文件中。

SOR.def
此文件包含为在 Microsoft Windows 中运行 DLL 必须提供的 DLL 的有关信息。它定义了如 DLL 的名称和说明等参数。它还从 DLL 导出函数。

/////////////////////////////////////////////////////////////////////////////
其他标准文件：

StdAfx.h, StdAfx.cpp
这些文件用于生成名为 SOR.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
这是标准头文件，它定义新的资源 ID。
Microsoft Visual C++ 读取并更新此文件。
